# Role Brief – Support & Concierge Lead

## Mission
Deliver the $50 “hand-hold” onboarding experience, resolve customer issues quickly, and channel insights back into documentation and automation.

## Inputs Provided
- Customer onboarding outline.
- Pricing structure + restock policy.
- Support contact channels (Telegram/Signal/email TBD).

## Key Deliverables
1. **Concierge Call Playbook**
   - 30–45 min agenda, checklist, and required info (API keys, channel setup).
   - Shared notes template to capture outcomes + next steps.
2. **Troubleshooting Flowchart**
   - Paths for “no response,” “API key rejected,” “Telegram relay not attaching,” etc.
3. **Feedback Loop Reports**
   - Weekly summary of customer pain points + recommended fixes.
   - Tag each issue to doc/automation owners.
4. **Knowledge Base Updates**
   - Work with Customer Writer to keep guides current.
   - Publish quick tip updates when patterns emerge.

## Success Metrics
- Concierge CSAT/NPS ≥8/10.
- Average response time <12h (until automated support grows).
- Return rate triggered by onboarding issues <5%.

## Collaboration
- Feed Customer Writer top questions for doc updates.
- Escalate technical blockers to Automation Engineer.
- Provide Ops Analyst with data on support volume + root causes.

## Reporting Cadence
- Daily log of support tickets/calls.
- Weekly insight report with prioritized fixes.
