# Role Brief – Device Prep Specialist

## Mission
Reduce hands-on configuration time for the Motorola XT2513v to ≤60 minutes (prep) and ≤60 minutes (reset) while ensuring every shipped unit passes QA on the first try.

## Inputs Provided
- `phone-prep-checklist.md` (baseline process)
- Financial targets (see `financials.md`)
- Team blueprint (`team-plan.md`)

## Key Deliverables
1. **Versioned SOP**
   - Step-by-step with timestamps, required tools, common failure modes.
   - Separate flows for new prep vs. return reset.
2. **QA Log Template + Samples**
   - Capture device ID, prep duration, issues found/fixed.
   - Provide 5 filled examples from initial batch.
3. **Golden Image Process Notes**
   - How to create/refresh the reference configuration.
   - Validation checklist before cloning.

## Success Metrics
- Avg prep time ≤60 min within 2 weeks.
- Defect rate <2% (no DOA complaints).
- 100% QA logs submitted within 24h of shipment.

## Collaboration
- Work with Automation Engineer to script repeatable steps.
- Feed timing data to Ops Analyst weekly.
- Surface any tooling requests immediately.

## Reporting Cadence
- Daily async update (time per unit, blockers).
- Weekly summary: trends, experiments, process changes.
